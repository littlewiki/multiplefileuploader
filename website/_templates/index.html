Welcome

MFU is a javascript/AJAX file uploader that provides seamless integration with dojotoolkit.
It enables user to have straightforward and really easy to use file uploader onto his web application and allows you to upload multiple files in a same time as gmail uploading system.

  • Cross-browser compliant: MFU has been successfully tested on IE6, IE7, firefox, Safari, chrome, Opera
  • Easy-to-use: MFU package provides samples to show you how easy is to put uploader in place in your web site
  • Templatable control: MFU uses the dojo template mechanism. As a matter of fact, the layout such as links, progress bar color/size and placement are completly configurable through html in widget directory
  • Internationalizable (i18n): All links and error messages are internationalizable through nls folder
  • Open source: MFU is distributed under BSD licence (same as dojo licence), so you can modify it in order to customize it for your web site
